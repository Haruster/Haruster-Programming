# Switch문

- 값에 따라 서로 다른 코드를 실행할 때 Switch문을 사용한다.

    
        switch (식) {

            case 값1: // 식의 결과가 값 1과 같을 때
                실행문장 1;
                break;

            case 값2: // 식의 결과가 값 2와 같을 때
                실행문장 2;
                break;

            case 값3: // 식의 결과가 값 3과 같을 때
                실행문장 3;
                break;

            default: // 어느 값도 일치하지 않을 때
                실행문장;

        }

    
## case 문의 '값'

- case문의 '값'은 상수(리터럴)만 가능하다.  

        case 1:
        case 2.7:
        case 'seoul':
        case true:

    
- case문의 '값'에 변수나 식은 사용이 불가하다.

        case a: 
        case a > 3:
