# Date 객체

- 시간 정보를 담는 객체

- 현재 시간 정보

        var now = new Date(); // 현재 날짜와 시간(시, 분, 초) 값으로 초기화된 객체 생성

- 학기 시작일 2017년 3월 1일의 날짜 기억

        var startDay = new Date(2017, 2, 1);    // 2017년 3월 1일 (2는 3을 뜻한다.)
