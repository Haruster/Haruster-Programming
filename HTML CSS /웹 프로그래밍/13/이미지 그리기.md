# 이미지 그리기

- 이미지 객체 생성

    > 파일에서 읽은 이미지를 담을 객체가 필요하다

        var img = new Image();


- 이미지 그리는 과정

    > 이미지 로딩과 onload

        - 이미지 파일의 로딩이 완료된 후 이미지를 그린다.

            img.onload = function() {

                .....   // img 객체에 로드된 이미지를 그린다.

            }

            img.src = "test.png";   // img 객체에 test.png 파일로부터 이미지의 로딩 시
