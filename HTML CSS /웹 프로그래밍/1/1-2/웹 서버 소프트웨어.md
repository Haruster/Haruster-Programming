# 웹 서버 소프트웨어


## 웹 서버 소프트웨어의 기능

    > 웹 브라우저로부터 요청(웹 문서 혹은 검색) 해석
    > 필요한 웹 서버 응용 프로그램 작동시키기
    > 웹 서버 응용프로그램의 실행 결과를 웹 브라우저로 전송


## 웹 서버 소프트웨어의 종류

    > Apache 사에서 만든 Apache
    > 마이크로 소프트 사에서 만들고 Windows NT에서만 실행되는 IIS
    > NGINX 사에서 만든 nginx
    > 구글에서 만들고 구글 사이트에서 실행되는 GWS(Google Web Server)
