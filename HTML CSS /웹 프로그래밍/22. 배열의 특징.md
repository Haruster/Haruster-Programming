# 배열의 특징

- 배열은 Array 객체이다.
    > []로 생성하더라도 Array 객체로 다루어진다. 


- 배열에 여러 타입의 데이터를 섞어서 저장할 수 있다.

        var any = new Array(5);     // 5개의 원소를 가진 배열 생성

        any[0] = 0;
        any[1] = 5.5;
        any[2] = "이미지 벡터"; // 문자열 저장
        any[3] = new Date();    // Date 객체 저장
        any[4] = convertFunction;   // function convertFunction()의 주소 저장
