# 문자열 연산자

- 문자열 연결 (+, +=)

ex)

    'abc' + 'dc'
    'abc' + 23
    

- 순서에 유의

    ex)

        23 + 53 + "abc" // 76abc로 출력되기 때문에 순서에 유의해서 연결하는 것이 좋다.


- 문자열 비교

    > 비교 연산자(!=, ==, >, <, <=, >=)는 문자열 비교에 사용된다.
    > 사전 순으로 비교 결과를 리턴한다.

    ex)

        var name = "haru"
        var ret = (name == "rayser"); // 비교 결과 false이다.
        var res = (name < "zero"); // namedl "zero"보다 앞에 나오므로 true이다.
