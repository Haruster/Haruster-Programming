# 지역 변수와 전역 변수

- 지역 변수 : 선언된 함수 내에서만 사용가능한 함수 (함수내에 var로 선언한다.)

- 전역 변수 : 프로그램 전역에서 사용가능한 함수 (함수 밖에 선언되거나 함수 내에 var 키워드 없이 선언된 변수이다.)


ex) 

    var x; // 전역 변수 x

    function f() {

        var y; // 지역변수 y선언

        x = 10; // 전역 변수 x에 10저장

        y = 10; // 지역 변수 y 초기화

    }


## this로 전역 변수 접근

- 지역 변수와 전역 변수의 이름이 같을 때
    > 전역 변수에 접근하고자 할 때 : this.전역변수


    ex) 

        var x; // 전역 변수

        function f() {

            var x; // 지역 변수

            x = 1; // 지역 변수 x에 1 저장

            this.x = 1000; // 전역 변수 x에 1000저장

        }
