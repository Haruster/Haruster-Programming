# 웹 페이지에서 자바스크립트의 역할 

## 사용자의 입력 및 계산

- 마우스와 키보드 입력은 오직 자바스크립트로만 가능하다.

- 계산이 가능하다


## 웹 페이지의 내용 및 모양의 동적제어가 가능하다

- HTML 태그의 속성, 콘텐츠, CSS 프로퍼티 값 동적 변경이 가능하다


## 브라우저 제어

- 브라우저 윈도우 크기와 모양 제어

- 새 윈도우 열기 / 닫기

- 다른 웹 사이트 접속

- 히스토리 제어
