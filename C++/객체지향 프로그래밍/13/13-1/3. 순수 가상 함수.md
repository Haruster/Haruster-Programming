# 순수 가상 함수

- 기본 클래스의 가상 함수 목적

    > 파생 클래스에서 재정의할 함수를 알려주는 역할

        - 실행할 코드를 작성할 목적이 아님


- 순수 가상 함수

    > pure virtual function

    > 함수의 코드가 없고, 선언만 있는 가상 멤버 함수


    > 선언 방법

        - 멤버 함수의 원형 = ();으로 선언

            class Shape {

            public:

                virtual void draw() = 0;    // 순수 가상 함수 선언

            }
