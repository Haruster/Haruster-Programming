# 객체 치환 및 객체 리턴

## 객체 치환

- 동일한 클래스 타입의 객체끼리 치환이 가능하다.

- 객체의 모든 데이터가 비트 단위로 복사된다.

        Circle c1(5);
        Circle c2(30);

        c1 = c2; // c2객체를 c1객체에 비트 단위 복사. c1의 반지름 30

> 치환된 두 객체는 현재 내용물만 같을 뿐 독립적인 공간을 유지하고 있다.


## 객체 리턴

- 객체의 복사본 리턴

        Circle getCircle() {

            Circle tmp(30);

            return tmp; // 객체 tmp리턴

        }

        Circle c;   // c의 반지름 1

        c = getCircle(); //tmp 객체의 복사본이 c에 치환;   c의 반지름 30
