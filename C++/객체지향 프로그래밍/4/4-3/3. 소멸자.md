# 소멸자

- 객제가 소멸되는 시점에서 자동으로 호출되는 함수이다.
    > 오직 한번만 자동 호출되며, 임의로 호출할 수는 없다.
    > 객체 메모리 소멸 직전 호출된다.

ex)
    class Circle {

        Circle();
        Circle(int r);

        ....

        ~Circle(); // 소멸자 함수 선언(리턴 타입도 없으며 매개 변수도 없고, 소멸자는 오직 하나만 존재한다.)

    };

    // 소멸자 함수 구현

    Circle :: ~Circle() {

        .......

    }
