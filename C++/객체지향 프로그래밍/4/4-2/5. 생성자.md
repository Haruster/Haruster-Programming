# 생성자

## 생성자(constructor)

- 객체가 생성되는 시점에서 자동으로 호출되는 멤버 함수이다.
- 클래스 이름과 동일한 멤버함수이다.

ex)
    class Circle {

        ....

        // 2개의 생성자 중복 선언
        Circle(); // 클래스 이름과 동일
        circle(int r); // 리턴타입을 명기하지 않음
        
        ...

    };

    // 생성자 함수 구현
    Circle :: Circle() { // 매개변수 없는 생성자
    
        ....

    }

    // 생성자 함수 구현
    Circle::Circle(int r) { // 매개변수를 가진 생성자
    
        ....

    }
