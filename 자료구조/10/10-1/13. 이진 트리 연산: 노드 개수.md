# 이진 트리 연산 : 노드 개수

- 참색 트리안의 노드의 개수를 계산한다.

- 각각의 서브트리에 대하여 순환 호출한 다음, 반환되는 값에 1을 더하여 반환한다.


        int get_count(TreeNode *node) {

            int count = 0;

            if(node != NULL) {

                count = 1 + get_node_count(node->left) + get_node_count(node->right);

            }

            return count;
        
        }
