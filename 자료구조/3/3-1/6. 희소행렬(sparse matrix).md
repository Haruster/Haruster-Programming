# 희소 행렬 (sparse matrix)

- 배열을 이용하여 행렬(matrix)를 표현하는 2가지 방법

    1. 2차원 배열을 이용하여 배열의 전체 요소를 지정하는 방법
    2. 0이 아닌 요소들만 저장하는 방법

- 방법 2가 더 유용하다.

> 희소 행렬 : 대부분의 항들이 0인 배열


                        0 0 0 7 0 0
                        9 0 0 0 0 8
         2 3 0          0 0 0 0 0 0
    A =  8 9 1    B  =  6 5 0 0 0 0
         7 0 5          0 0 0 0 0 1
                        0 0 2 0 0 0
