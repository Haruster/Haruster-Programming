# 기본 연산자

+ : 더하기 ex) 1 + 2

- : 빼기 ex) 2 - 1

* : 곱하기 ex) 2 * 3

/ : 나누기 ex) 6 / 2

% : 나머지 ex) 5 % 3

** : 거듭제곱 ex) 2 ** 3


## 축약 연산

+= : 더하기

-= : 빼기

*= : 곱하기

/= : 나누기

%= : 나머지


## 증감 연산자

++ : +1 증가 (선증가는 그 줄에서 값을 증가시키고, 후증가는 그 줄이 끝난 다음에 증가를 시킨다.)

-- : -1 감소 (선감소는 그 줄에서 값을 감소시키고, 후감소는 그 줄이 끝난 다음에 감소를 시킨다.)


## 비교 연산자

< : 작다.

> : 크다

<= : 작거나 같다.

>= : 크거나 같다.

== : 같다.

!= : 다르다.

=== : 동등연산자로 두 값의 타입을 비교한다.
